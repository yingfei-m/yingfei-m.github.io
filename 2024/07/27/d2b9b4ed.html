git 基础配置

登陆 Github 找到 Setting 页面，选择左边的 点击左边的“SSH and GPG keys”，然后点击右边的“New SSH Key” 如下图所示

把刚才复制的公钥贴在 Key 的框框里面，同时在 Title 的框框里给这个公钥取一个名字，方便和其他的公钥区分，然后点击 Add SSH key 按钮即可添加成功
